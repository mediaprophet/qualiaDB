⚠️ Full Disclosure
Numbers are produced by real JavaScript executing now. No hardcoded values exist. The harness attempts to load Oxigraph WASM directly from the npm CDN (esm.sh). If it fails (CORS, CDN downtime, module API mismatch), the harness clearly marks those cells as LOAD FAILED and falls back to a BTreeMap-equivalent proxy — labelled as such.
