Methodology: Latency figures from live benchmark harness (n=30 samples, 10 warmup) on 10K synthetic N-Triples, Ubuntu 22.04, GitHub Actions runner. Qualia: HTTP daemon loopback (port 4242, dev mode). Oxigraph: pyoxigraph in-process + 0.3ms HTTP estimate. SurrealDB: HTTP (surreal binary). Comunica: Node.js in-process + 0.3ms HTTP estimate. QPS = 1000 / p50_ms × vCPU × 0.65 parallelism factor. RAM scales linearly from 10K-triple baseline. Cloud pricing: AWS us-east-1 on-demand, June 2026 estimates. On-premise: hardware amortised over 10 years + 15% power/cooling. All figures are estimates — run python benchmarks/harness.py --all to generate real numbers for your environment.
