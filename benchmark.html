Native Local Daemon

To bypass the browser's WASM Sandbox and execute bare-metal Rust queries natively over the local loopback WebSocket, clone the repository and boot the daemon:

git clone https://github.com/mediaprophet/qualiaDB.git

cd qualiaDB

cargo run --release -p qualia-cli -- daemon --port 4242
