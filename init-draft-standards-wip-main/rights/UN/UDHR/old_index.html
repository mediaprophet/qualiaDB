Abstract

This document specifies an upper-level ontology for UN Human Rights Instruments, including the Universal Declaration of Human Rights (UDHR), Convention on the Rights of the Child (CRC), International Covenant on Civil and Political Rights (ICCPR), and others. It provides a structured, machine-readable framework for modeling provisions, rights, violations, and protection mechanisms, with specific instrument provisions represented as literals. The ontology is extensible and supports semantic web applications such as reasoning and querying.

Status of This Document

This is an unofficial ontology specification for UN Human Rights Instruments, created for demonstration purposes. It is not endorsed by the United Nations or UNESCO.

Introduction

The UN Human Rights Instruments ontology provides a generalized framework for modeling human rights documents listed by the UN Office of the High Commissioner for Human Rights, including treaties, conventions, and declarations. It defines classes for instruments, provisions, rights, violations, and protection mechanisms, enabling interoperability and extensibility for specific instruments such as the UDHR, CRC, and ICCPR.

Namespace

The namespace for this ontology is http://www.udhr.org/ontology/udhr#. Prefixes used include:

  • udhr: http://www.udhr.org/ontology/udhr#
  • rdf: http://www.w3.org/1999/02/22-rdf-syntax-ns#
  • rdfs: http://www.w3.org/2000/01/rdf-schema#
  • owl: http://www.w3.org/2002/07/owl#
  • xsd: http://www.w3.org/2001/XMLSchema#

Ontology Structure

Classes

The ontology defines the following classes:

Class Description URI
HumanRightsInstrument A UN human rights document (e.g., UDHR, ICCPR, CRC). udhr:HumanRightsInstrument
Provision An article or clause within an instrument. udhr:Article
Right A human right defined by an instrument. udhr:HumanRights
Violation An act or omission contravening a right. udhr:ActOmission
ProtectionMechanism A mechanism to promote or protect rights. udhr:HumanRightsProtectionMechanism
Person An individual entitled to rights. udhr:Person
Group A collective entitled to rights. udhr:Group
Child A person under a certain age, with specific rights. udhr:Child
Discrimination A violation involving unfair treatment based on specific grounds. udhr:Discrimination
ChildEducation Right to education for children. udhr:ChildEducation

Properties

The ontology includes the following properties:

Property Type Domain Range Description
udhr:hasInstrumentName Data Property udhr:HumanRightsInstrument xsd:string Specifies the name of the instrument (e.g., "UDHR").
udhr:hasProvision Object Property udhr:HumanRightsInstrument udhr:Article Links an instrument to its provisions.
udhr:hasArticleNumber Data Property udhr:Article xsd:integer Specifies the article or clause number.
udhr:hasContent Data Property udhr:Article xsd:string Contains the textual content of a provision.
udhr:definesRight Object Property udhr:Article udhr:HumanRights Links a provision to the right it defines.
udhr:appliesTo Object Property udhr:HumanRights udhr:Person or udhr:Group or udhr:Child Links a right to its entitled entities.
udhr:addressesViolation Object Property udhr:Article udhr:ActOmission Links a provision to a violation it addresses.
udhr:hasProtectionMechanism Object Property udhr:Article or udhr:ActOmission udhr:HumanRightsProtectionMechanism Links a provision or violation to a protection mechanism.

Constraints and Axioms

The ontology enforces the following constraints:

  • Each udhr:Article must have exactly one udhr:hasArticleNumber.
  • udhr:Person, udhr:Group, and udhr:Child are disjoint classes.
  • Each udhr:HumanRightsInstrument must have at least one udhr:hasProvision.

Usage Examples

SPARQL query to retrieve provisions addressing specific violations across instruments:

PREFIX udhr: 
SELECT ?instrument ?provision ?content ?violation
WHERE {
    ?instrument a udhr:HumanRightsInstrument ;
                udhr:hasProvision ?provision .
    ?provision udhr:hasContent ?content ;
               udhr:addressesViolation ?violation .
    ?violation a udhr:ActOmission .
}

SPARQL query to find rights and protection mechanisms for children:

PREFIX udhr: 
SELECT ?right ?mechanism
WHERE {
    ?right a udhr:HumanRights ;
           udhr:appliesTo udhr:Child ;
           udhr:hasProtectionMechanism ?mechanism .
}
