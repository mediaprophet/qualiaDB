# Qualia Icons

This folder contains icon assets for the Qualia ecosystem.

## Current Status (June 2026)

- **Qualia / QualiaDB**: Primary direction is the Q-Star + 42 concept (Option 1).
- **Webizen**: Primary direction is the quantum glowing orb / guardian style.

These assets are prepared early so the team can refine nomenclature and product strategy without friction.

The broader ecosystem (Wellfair, Webizen, Cooperative Projects, ontology) is intended to act as a protective, human-centric layer around the core QualiaDB engine.

## Files

- `qualia/` — Qualia / QualiaDB brand icons
- `webizen/` — Webizen (personal guardian/agent) icons

## Notes

- Icons are currently high-resolution PNGs.
- Multi-size exports and .ico generation can be added once final directions are locked.
- Feel free to reorganize or rename as product strategy evolves.
